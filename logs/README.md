# Logs

This folder stores role-based work summaries after an issue is closed.

## Directories

- `architect/`
- `developer/`
- `qa/`
- `ops-sre/`
- `security/`

## Naming

Recommended filename format:
- `issue-<number>-<short-slug>.md`

## Suggested Content

- Context
- Actions taken
- Key decisions
- Risks or follow-up items
- Lessons learned
- Collaboration improvements for future work
